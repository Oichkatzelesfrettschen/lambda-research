# Research Topic Tags

This page lists all tags used throughout the lambda calculus research repository for easy navigation and cross-referencing.

## Core Lambda Calculi Tags
- `untyped-lambda` - Church's original untyped lambda calculus
- `simply-typed` - Simply typed lambda calculus
- `system-f` - Polymorphic lambda calculus (System F)
- `dependent-types` - Types depending on values
- `linear-types` - Resource-aware computation

## Advanced Type System Tags
- `session-types` - Communication protocol types
- `effect-systems` - Computational effects and algebraic effects
- `gradual-typing` - Dynamic-static typing integration
- `refinement-types` - Types with logical predicates
- `intersection-types` - Types for normalization properties
- `union-types` - Discriminated unions and sum types

## Theoretical Foundation Tags
- `category-theory` - Categorical semantics and foundations
- `proof-theory` - Formal study of mathematical proofs
- `domain-theory` - Mathematical semantics for computation
- `homotopy-type-theory` - Univalent foundations for mathematics
- `combinatory-logic` - Combinatorial basis for lambda calculus

## Implementation Tags
- `rust-impl` - Rust language implementations
- `haskell-impl` - Haskell language implementations
- `agda-impl` - Agda implementations with formal verification
- `coq-impl` - Coq implementations with proofs
- `ocaml-impl` - OCaml implementations

## Research Period Tags
- `historical` - Foundational papers (pre-1980)
- `classical` - Classical works (1980-2000)
- `modern` - Modern developments (2000-2020)
- `recent` - Recent research (2020+)

## Application Domain Tags
- `quantum-computing` - Quantum lambda calculi and quantum types
- `probabilistic` - Probabilistic programming and uncertainty
- `concurrent` - Pi-calculus, actor models, concurrent variants
- `functional-programming` - Practical functional programming applications

## Paper Type Tags
- `survey` - Survey and tutorial papers
- `implementation` - Implementation-focused papers
- `theory` - Pure theoretical contributions
- `applications` - Applied research and case studies
