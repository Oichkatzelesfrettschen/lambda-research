#### **Church's Lambda Calculus (1936)**
- **Impact**: Foundation of modern computational theory
- **Key Insight**: Effective computability through function abstraction
- **Legacy**: Basis for functional programming languages and proof assistants

#### **Curry-Howard Correspondence**
- **Discovery**: Deep connection between proofs and programs
- **Significance**: Types correspond to propositions, programs to proofs
- **Applications**: Modern dependently typed languages (Agda, Idris, Lean)

#### **System F Polymorphism (Girard-Reynolds, 1972-1974)**
- **Innovation**: Parametric polymorphism with second-order quantification
- **Theoretical Impact**: Strong normalization and parametricity theorems
- **Practical Impact**: Generic programming in Haskell, ML, and modern languages
