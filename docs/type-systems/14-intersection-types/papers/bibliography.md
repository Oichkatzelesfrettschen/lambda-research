# Intersection Types: Comprehensive Academic Bibliography

### Coppo-Dezani Original Work

**Coppo, M. and Dezani, M.** (1978). *A New Type-Assignment for Lambda Terms*. Archiv für Mathematische Logik, 19, 139-156.
- **Key Contribution**: First introduction of intersection types for lambda calculus
- **Theoretical Significance**: Characterized strongly normalizing lambda terms through type assignment
- **DOI**: 10.1007/BF02011875

**Coppo, M. and Dezani, M.** (1980). *An Extension of the Basic Functionality Theory for the λ-Calculus*. Notre Dame Journal of Formal Logic, 21, 685-693.
- **Key Contribution**: Extended intersection type theory with additional properties
- **Practical Applications**: Semantic analysis of lambda terms
- **DOI**: 10.1305/ndjfl/1093883253

**Coppo, M., Dezani, M. and Venneri, B.** (1980). *Principal Type Schemes and λ-Calculus Semantics*. In: To H. B. Curry, J. R. Hindley and J. P. Seldin (eds), Academic Press, pp. 535-560.
- **Key Contribution**: Principal type schemes for intersection types
- **Connections**: Links to polymorphism theory



### BCD System Development

**Barendregt, H., Coppo, M., and Dezani-Ciancaglini, M.** (1983). *A filter lambda model and the completeness of type assignment*. Journal of Symbolic Logic, 48(4), 931-940.
- **Key Contribution**: The BCD system - most quoted intersection type system
- **Theoretical Significance**: Filter lambda-model semantics for intersection types
- **Connections**: Foundational for lambda-calculus semantics
- **DOI**: 10.2307/2273659


Principal Type Inference and Decidability



### Kfoury-Tiuryn Contributions

**Kfoury, A.J. and Tiuryn, J.** (1992). *Type reconstruction in finite rank fragments of second-order lambda calculus*. Information and Computation, 98(2), 228-257.
- **Key Contribution**: Principal type inference for rank-restricted intersection types
- **Theoretical Significance**: Decidability results for finite-rank systems
- **DOI**: 10.1016/0890-5401(92)90028-V

**Kfoury, A.J., Tiuryn, J. and Urzyczyn, P.** (1990). *The undecidability of the semi-unification problem*. In: Proceedings of the 22nd Annual ACM Symposium on Theory of Computing, pp. 468-476.
- **Key Contribution**: Complexity theoretic results for intersection type inference
- **Theoretical Significance**: DEXPTIME-completeness of ML type inference

**Kfoury, A.J. and Wells, J.B.** (2004). *Principality and type inference for intersection types using expansion variables*. Theoretical Computer Science, 311(1-3), 1-70.
- **Key Contribution**: Expansion variables for intersection type inference
- **Practical Applications**: Compositional analysis methods
- **DOI**: 10.1016/S0304-3975(03)00577-2


Strict Intersection Types



### van Bakel's System

**van Bakel, S.** (1992). *Complete restrictions of the Intersection Type Discipline*. Theoretical Computer Science, 102, 135-163.
- **Key Contribution**: Strict intersection type assignment system
- **Theoretical Significance**: Syntax-directed system with BCD properties
- **DOI**: 10.1016/0304-3975(92)90297-S

**van Bakel, S.** (2011). *Strict Intersection Types for the Lambda Calculus*. ACM Computing Surveys, 43(3), Article 20.
- **Key Contribution**: Comprehensive survey of strict intersection types
- **Practical Applications**: Characterization of normalization properties
- **DOI**: 10.1145/1922649.1922657

**van Bakel, S.** (1988). *Essential Intersection Type Assignment*. Master's thesis.
- **Key Contribution**: First version of strict type assignment system
- **Historical Significance**: Coined the term "strict" for intersection types


Hindley's Contributions



### Simple Types with Intersection

**Hindley, J.R.** (1992). *Types with intersection: An introduction*. Formal Aspects of Computing, 4, 470-486.
- **Key Contribution**: Informal introduction to intersection type theory
- **Educational Value**: Accessible introduction to intersection connective
- **Connections**: Universal type and function-types
- **DOI**: 10.1007/BF01211394

**Hindley, J.R.** (1997). *Basic Simple Type Theory*. Cambridge Tracts in Theoretical Computer Science, No. 42. Cambridge University Press.
- **Key Contribution**: Foundational text on simple type theory
- **Educational Value**: Comprehensive treatment of type systems
- **ISBN**: 978-0521054225


Ronchi della Rocca's Semantic Work



### Compositional Characterizations

**Ronchi Della Rocca, S.** (2002). *Compositional Characterizations of λ-Terms Using Intersection Types*. In: Theoretical Computer Science, ICALP 2002. Lecture Notes in Computer Science, vol 2380. Springer.
- **Key Contribution**: Compositional methods for evaluation properties
- **Theoretical Significance**: Set-theoretical semantics over stable sets
- **DOI**: 10.1007/3-540-45465-9_26

**Egidi, L., Honsell, F., and Ronchi Della Rocca, S.** (1992). *Operational, denotational and logical descriptions: a case study*. Fundamenta Informaticae, 16(2), 149-169.
- **Key Contribution**: Multiple semantic perspectives on lambda calculus
- **Connections**: Intersection types and operational semantics

**Honsell, F. and Ronchi Della Rocca, S.** (1992). *An approximation theorem for topological lambda models and the topological incompleteness of lambda calculus*. Journal of Computer and System Sciences, 45(1), 49-75.
- **Key Contribution**: Topological aspects of lambda models
- **Theoretical Significance**: Incompleteness results
- **DOI**: 10.1016/0022-0000(92)90052-8


Modern Developments and Extensions

**van Bakel, S. and Fernández, M.** (2017). *Intersection Types for the lambda-mu Calculus*. Logical Methods in Computer Science, 13(4).

- **Theoretical Significance**: Subject reduction and expansion invariance
- **DOI**: 10.23638/LMCS-13(4:4)2017

**Bakel, S. van and Fernández, M.** (2017). *Characterisation of Approximation and (Head) Normalisation for λμ using Strict Intersection Types*. arXiv:1702.02273.

- **Practical Applications**: Head normal form analysis

Summary of Key Properties



### Theoretical Properties
1. **Strong Normalization**: Intersection types characterize strongly normalizing terms
2. **Principal Types**: Finite-rank restrictions have principal typings
3. **Decidability**: Type inference is decidable for finite-rank systems
4. **Completeness**: Complete characterization using filter semantics



### Practical Applications
1. **Programming Languages**: TypeScript, Flow, Scala 3
2. **Type Inference**: Bidirectional type checking algorithms
3. **Program Analysis**: Compositional evaluation property characterization
4. **Semantic Analysis**: Lambda calculus and functional programming



### Open Research Directions
1. **Probabilistic Extensions**: Almost-sure termination analysis
2. **Control Operators**: Classical lambda calculus extensions
3. **Performance**: Efficient type inference algorithms
4. **Integration**: Mainstream programming language adoption
