# Comprehensive Academic Bibliography: Session Types (1993-2025)

**Honda, K.** (1993). *Types for Dyadic Interaction*. Venue information needed.
- **Key Contribution**: Contribution:** Introduces the foundational concept of session types as a typed formalism for concurrency where types denote freely composable structure of dyadic interaction. This paper formulates the first session type system for structured communication protocols.

**Honda, K.** (1998). *Language Primitives and Type Discipline for Structured Communication-Based Programming*. Venue information needed.
- **Key Contribution**: Contribution:** Establishes basic language constructs and type discipline as foundation for structured communication-based concurrent programming. Introduces syntactic structure exploited by ML-style type discipline for high-level type abstraction of interactive behaviors.

**Gay, S.J.a.M.H.** (1999). *Types and Subtypes for Client-Server Interactions*. Venue information needed.
- **Key Contribution**: Contribution:** Early work on subtyping relationships for session types in client-server communication patterns.

**Gay, S.J.a.M.H.** (2005). *Subtyping for Session Types in the Pi Calculus*. Venue information needed.
- **Key Contribution**: Contribution:** Foundational work establishing subtyping relationships for session types in the pi-calculus, enabling safe substitution of communication protocols.

**Neubauer, M.a.P.T.** (2004). *An Implementation of Session Types*. Venue information needed.
- **Key Contribution**: Contribution:** First practical implementation of session types in Haskell, bridging theory and practice for functional programming languages.

**Honda, K.** (2008). *Multiparty Asynchronous Session Types*. Venue information needed.
- **Key Contribution**: Contribution:** Introduces multiparty session types extending binary sessions to n-party asynchronous interactions. Establishes global types and endpoint projection as fundamental concepts.

**Honda, K.** (2016). *Multiparty Asynchronous Session Types*. Venue information needed.
- **Key Contribution**: Contribution:** Extended journal version providing comprehensive theory of multiparty session types with detailed proofs of communication safety, progress, and session fidelity.

**Yoshida, N.** (2013). *The Scribble Protocol Language*. Venue information needed.
- **Key Contribution**: Contribution:** Introduces Scribble as a practical protocol description language based on multiparty session types theory, enabling industrial applications.

**Gay, S.J.a.V.T.V.** (2010). *Linear Type Theory for Asynchronous Session Types*. Venue information needed.
- **Key Contribution**: Contribution:** Establishes deep connections between session types and linear logic, providing theoretical foundations for resource-aware concurrent programming.

**Wadler, P.** (2012). *Propositions as Sessions*. Venue information needed.
- **Key Contribution**: Contribution:** Demonstrates Curry-Howard correspondence between propositions of linear logic and session types, extending classical type theory to concurrent computation.

**Caires, L.a.F.P.** (2010). *Session Types as Intuitionistic Linear Propositions*. Venue information needed.
- **Key Contribution**: Contribution:** Establishes formal connection between session types and intuitionistic linear logic, providing logical foundations for concurrent process communication.

**Toninho, B.** (2011). *Dependent Session Types via Intuitionistic Linear Type Theory*. Venue information needed.
- **Key Contribution**: Contribution:** Extends session types with dependent types, enabling rich constraints on sessions such as interface contracts and proof-carrying certification.

**Ancona, D.** (2016). *Behavioral Types in Programming Languages*. Venue information needed.
- **Key Contribution**: Contribution:** Comprehensive survey of behavioral types encompassing interfaces, communication protocols, contracts, and choreography. Provides definitive overview of practical aspects and pragmatics.

**Gay, S.J.** (2016). *Subtyping Supports Safe Session Substitution*. Venue information needed.
- **Key Contribution**: Contribution:** Demonstrates how subtyping enables safe substitution of session types while preserving communication protocols and safety properties.

**Lindley, S.a.J.G.M.** (2015). *A Semantics for Propositions as Sessions*. Venue information needed.
- **Key Contribution**: Contribution:** Provides operational semantics for session-typed functional language GV, demonstrating deadlock-freedom, determinism, and termination properties.

**Ng, N.** (2012). *Multiparty Session C: Safe Parallel Programming with Message Optimisation*. Venue information needed.
- **Key Contribution**: Contribution:** Implementation of multiparty session types in C programming language with optimizations for parallel programming and message passing.

**Fowler, S.** (2019). *Exceptional Asynchronous Session Types: Session Types without Tiers*. Venue information needed.
- **Key Contribution**: Contribution:** Extends Links programming language with asynchronous session types, enabling tierless web programming with strong communication guarantees.

**Kokke, W.a.O.D.** (2021). *Deadlock-Free Session Types in Linear Haskell*. Venue information needed.
- **Key Contribution**: Contribution:** Implements session types in Linear Haskell with priority-based deadlock freedom, providing practical library for concurrent programming.

**Padovani, L.** (2017). *A Simple Library Implementation of Binary Sessions*. Venue information needed.
- **Key Contribution**: Contribution:** Demonstrates practical implementation techniques for ensuring deadlock freedom in binary session types through careful library design.

**Dardha, O.a.S.J.G.** (2018). *A New Linear Logic for Deadlock-Free Session-Typed Processes*. Venue information needed.
- **Key Contribution**: Contribution:** Introduces novel linear logic that ensures deadlock freedom for session-typed processes without additional syntactic restrictions.

**Balzer, S.a.F.P.** (2017). *Manifest Sharing with Session Types*. Venue information needed.
- **Key Contribution**: Contribution:** Addresses sharing of resources in session-typed programs while maintaining linearity constraints and deadlock freedom.

**Mostrous, D.a.V.T.V.** (2017). *Session Typing for a Featherweight Erlang*. Venue information needed.
- **Key Contribution**: Contribution:** Adapts session types to actor-based programming model in Erlang, handling asynchronous message passing and actor lifecycle.

**Jespersen, T.B.L.** (2015). *Session Types for Rust*. Venue information needed.
- **Key Contribution**: Contribution:** Implementation of session types in Rust leveraging ownership types for memory safety and communication protocol verification.

**Thiemann, P.a.V.T.V.** (2020). *Context-Free Session Type Inference*. Venue information needed.
- **Key Contribution**: Contribution:** Develops inference algorithms for session types using context-free grammar techniques, improving usability and automation.

**Castro-Perez, D.a.N.Y.** (2020). *CAMP: Cost-Aware Multiparty Session Protocols*. Venue information needed.
- **Key Contribution**: Contribution:** Extends multiparty session types with cost analysis, enabling reasoning about resource consumption in distributed protocols.

**Igarashi, A.** (2017). *Gradual Session Types*. Venue information needed.
- **Key Contribution**: Contribution:** Introduces gradual typing for session types, enabling seamless transition between static and dynamic typing in communication protocols.

**Dardha, O.** (2023). *Mechanizing Session-Types using a Structural View: Enforcing Linearity without Linearity*. Venue information needed.
- **Key Contribution**: Contribution:** Novel approach to mechanizing session types that localizes linearity conditions as predicates, enabling structural typing contexts instead of linear ones.

**Chen, H.** (2024). *Verified Lock-Free Session Channels with Linking*. Venue information needed.
- **Key Contribution**: Contribution:** Implements and verifies lock-free session channels with bidirectional communication and linking operations using separation logic.

**Golov, A.** (2024). *Probabilistic Refinement Session Types*. Venue information needed.
- **Key Contribution**: Contribution:** Introduces probabilistic refinement session types enabling symbolic reasoning for concurrent probabilistic systems, extending session types to randomized protocols.

**Padovani, L.** (2021). *Fair Asynchronous Session Subtyping*. Venue information needed.
- **Key Contribution**: Contribution:** Develops fair subtyping relation that preserves liveness properties in asynchronous session types, addressing limitations of traditional subtyping.

**Barbanera, F.** (2025). *A Sound and Complete Characterization of Fair Asynchronous Session Subtyping*. Journal publications: 9 papers (Journal of ACM, Journal of Functional Programming, Acta Informatica).
- **Key Contribution**: Contribution:** Provides first sound and complete characterization of fair asynchronous session refinement, solving long-standing open problem in session type theory.
